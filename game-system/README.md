# 🃏 Sistema de Regras — MXM (Marcos X Marcos)

Esta pasta reúne os documentos oficiais do sistema de regras utilizado nas campanhas do RPGxRPG.

## O que é o MXM?

O **MXM (Marcos X Marcos)** é um sistema de RPG de mesa criado por Marcos, ambientado no
universo de *Hunter x Hunter* de Yoshihiro Togashi. Foi construído sobre a base mecânica do
*Vampiro: A Máscara 5ª Edição* (White Wolf Publishing), adaptado para incorporar os conceitos
do universo HxH — principalmente o sistema de **Nen**, combate e progressão de personagem.

## 🏷️ Tags dos documentos

Os documentos seguem uma convenção de nomenclatura com prefixo indicando o conteúdo:

| Tag | Significado |
|---|---|
| `[REGRAS]` | Mecânicas de combate, resolução de ações e núcleo do sistema |
| `[FICHA]` | Criação de personagem, atributos, habilidades e progressão |
| `[CATALOGO]` | Armas, consumíveis e equipamentos |
| `[REF]` | Referência rápida para consulta durante a sessão |
| `[GUIA]` | Orientações para o Mestre e adaptações do sistema |
